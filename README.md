## ЁЯУЪ What's Inside?

This repository is packed with hands-on examples. Instead of just theory, every concept is backed by functional Java code. Here is a breakdown of the topics covered:

### 1. Java Basics & Data Structures
* **Arrays:** 1D and 2D arrays, matrix addition (`Addition_2D_Array.java`), and multidimensional arrays.
* **Algorithms:** Prefix sum implementation (`PrefixSum.java`).

### 2. Core OOP Concepts
* **Classes & Objects:** Creating blueprints and instantiating objects (`Student`, `Employee`, `Animal`, `Dog`).
* **Constructors:** Default, parameterized, and constructor overloading (`constructor.java`, `B5.java`).
* **Keywords:** Practical uses of `this` and `super` (`B14.java`, `B15.java`).
* **Blocks:** Understanding Instance vs. Static initialization blocks (`B8.java`).

### 3. Inheritance (The "IS-A" Relationship)
* **Simple Inheritance:** Basic parent-child class structures (`B9.java`).
* **Multilevel Inheritance:** Chains of inheritance (`B10.java`).
* **Hierarchical Inheritance:** Multiple child classes from a single parent (`B13.java`).
* **Multiple Inheritance:** Solving the diamond problem using Interfaces (`B12.java`, `B24.java`).

### 4. Polymorphism
* **Compile-time Polymorphism:** Method and Constructor Overloading.
* **Run-time Polymorphism:** Method Overriding using the `@Override` annotation (`B16.java`, `B29.java`).

### 5. Abstraction & Encapsulation
* **Encapsulation:** Data hiding and getter/setter methods (`B17.java`).
* **Abstract Classes:** Partial abstraction with `abstract` keywords (`B18.java`, `B19.java`, `B20.java`).
* **Interfaces:** 100% abstraction, including modern Java features like `default`, `static`, and `private` methods in interfaces (`B21.java` through `B28.java`).

    ```bash

    ```
